HTML基础知识整理

该文用于记录HTML大体的内容,目的在于需要时能够迅速的捡起所需的知识

1、什么是HTML：

​ Hyper Text Markup Language （超文本标记语言）

2、W3C标准（World Wide Web Consortium 万维网联盟）：

         1. 结构化标准语言（HTML,XML）
                  2. 表现标准语言（CSS）
                    3. 行为标准（DOM,ECMAScript）

3、HTML构成元素:

  • 块级元素:独占一行,可以设置宽高,默认宽度为浏览器宽度
  • 行内元素:可以多个标签放在一行,宽高属性不生效,依靠内容撑开
  • 行内块元素:结合行内元素和块级元素的优点,可以设置宽高,同时可以多个标签存在同一行
  • 各种标签之间的转换:
    1、块级标签转换为行内标签：display:inline;
    2、行内标签转换为块级标签：display:block;
    3、转换为行内块标签：display：inline-block;
